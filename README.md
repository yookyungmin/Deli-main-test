
# kh_final_deli
## 식당 운영자와 소비자를 위한 배달 웹앱 - 딜리 Deli

### 프로젝트 소개
코로나로 인해 배달 음식의 수요가 늘어남에 따라 배달 웹앱을 만들어 보고자 기획 되었습니다.<br>
팀원 : 7명
<br>
